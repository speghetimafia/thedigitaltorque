# The Digital Torque

A premium, editorial-style single page website.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design Notes

- **Typography**: Uses 'Pinyon Script' for headings and 'Lora' for body text.
- **Colors**: Deep crimson (#c02b2b) accents on white.
- **Images**: Located in `public/assets`.
- **Animations**: simple fade-in and slide-up effects using IntersectionObserver.
